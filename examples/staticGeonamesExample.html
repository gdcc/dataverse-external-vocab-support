Quick Static Geonames Demonstrator

This static (client side) HTML page demonstrates how the scripts in this repository are designed to attach to specific fields via the use of data-cvoc-* attributes embedded in the page. Once you've viewed the page and tried it out, take a look at the page source to see the added attributes. With Dataverse, these attributes are automatically generated based on the :CVocConf setting. For other repositories - if you can add these attributes to your page, you can use the scripts as well.
Note - The input is not connected to any output (display). If someone wants to add Javascript to add and update a display when the input is changed, feel free!)

This example uses the Geonames service for which you must have an account (username) and have the API usage enabled.
